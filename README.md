# 🖧 Network Design for XYZ Software Company
 
> A mini project on enterprise network design submitted to the Department of Electronics and Computer Engineering, Pulchowk Campus, Tribhuvan University.
 
---
 
## 📌 Overview
 
This project designs a complete computer network infrastructure for **XYZ Software Company** using industry-standard technologies including **OSPF routing**, **VLANs**, **DHCP**, **DNS**, and **VLSM-based IP subnetting**. The network is simulated using **Cisco Packet Tracer**.

## 🎯 Objectives
 
- Design an efficient routed network using **OSPF** (multi-area)
- Segment departments using **VLANs**
- Automate IP address assignment using **DHCP**
- Provide internet access via an **ISP router**
- Implement **DNS** and **Web servers** for internal services
 
---
## 🛠️ Technologies Used
 
- **Cisco Packet Tracer** — Network simulation
- **OSPF** — Dynamic routing protocol (multi-area)
- **VLSM** — Variable Length Subnet Masking for efficient IP allocation
- **VLAN** — Layer 2 network segmentation
- **DHCP** — Automatic IP address management
- **DNS** — Name resolution
- **HTTP/Web Server** — Internal web service
 
---

## ✅ Results
 
- DHCP successfully assigns IP addresses dynamically to all devices
- Full connectivity verified using `ping` and `tracert` commands
- All internal networks reachable from any host in the topology
- OSPF routing converges correctly across all three areas
